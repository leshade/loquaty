class Console ;
Console クラスはレガシー・スタイルなコンソールへの入出力関数を提供します。
コンソールは仮想ターミナル シーケンスが有効な状態に設定されています。

getch
public static int getch( )
キー入力が存在する場合、その文字コードを取得します。キー入力が存在しない場合はも即座に関数は復帰します。
Return Value
入力されたーコード。入力が存在しない場合は 0

public static String input( )
