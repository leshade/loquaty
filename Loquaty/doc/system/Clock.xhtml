Clock
Summary
Clock はシステム・クロックを取得するクラスです。
このクラスのクロックは DateTime のエポック時間とは互換性がありません。経過時間を測定することに利用できます。

timeZone
public static long timeZone( )
ローカルのタイムゾーンと UTC の時差を秒単位で取得します。
Clock.localDate() + (double) Clock.timeZone() == Clock.utcDate() の関係が成立します。
※実際に実行すると関数の実行に要する時間差によって厳密には == が成立しません。
