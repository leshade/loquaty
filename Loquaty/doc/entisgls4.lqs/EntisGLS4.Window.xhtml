lock
public native boolean lock( long msecTimeout = -1 )
ウィンドウスレッド排他処理用
msecTimeout
ミリ秒単位のタイムアウト時間。-1 の場合は成功するまで待ち続けます。
Return Value
排他処理権を獲得できた場合には true
lock() が true を返してから unlock() を呼び出すまでの間、ウィンドウのイベント処理はすべて停止します。
lock()～unlock() は対になっていなければなりませんが、再帰的（入れ子状）に呼び出すことができます。
