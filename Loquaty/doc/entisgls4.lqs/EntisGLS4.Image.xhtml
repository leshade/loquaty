EntisGLS4.Image
Summary
class Image ;
画像オブジェクト

Image オブジェクトは１枚、又は複数の画像を保持しますがselectFrame によって選択されている１枚の画像として扱われます。
selectFrame による作用が問題になる場合には newReference 関数で特定のフレームだけの Image オブジェクトを取得します。
