EntisGLS4.SceneCustomItem
Summary
class SceneCustomItem ;
スクリプト実装 Scene アイテム

シーン・コンポジションのアイテムとして機能し、また GUI エディタ上のインターフェースを提供します。

public なメンバ変数が以下の型の場合、xml 形式のコメントでGUI インターフェースを設定できます。
boolean, int, uint, double, ARGB8(RGB要素のみ), Vector3, Vector3d,
Matrix3, Matrix3d, Quaternion, Quaterniond, Vector2, Vector2d,
Vector4, Vector4d, Matrix4, Matrix4d,
String, Image, AudioPlayer, ModelBuffer, ModelPose, SceneItem

以下のタグでより詳細に設定できます。
<name> タグ
表示項目名（必須）
<id> タグ
保存時のパラメータ ID。省略時には変数名がそのまま ID となります。
（※ SceneItem.ParameterID や SceneCommon.ParameterID 列挙子と同じ名前は正常に機能しません）
<desc> タグ
詳細説明
<sub_type> タグ
GUI 上の型指定（省略した場合には対応するデフォルトの型が適用されます）
matrix, position, direction, zoom, rotation, color, scalar,
integer, boolean, selector, command, pose, matrix4, vector4, vector2
<range> タグ
scalar 値域指定。 min=最小値, max 最大値
<selector> タグ
GUI 上でドロップダウンリスト形式で入力。
int, String で使用可。
Image, AudioPlayer, ModelBuffer, ModelPose は自動的にドロップダウンリストになるため必要ありません。
<selector><enum> タグ
ドロップダウンリストの位置項目。 name=表示名, num=選択時の数値, str=選択時の文字列

// 例
class ExampleItem extends SceneCustomItem
{
    /*
    <name>色</name>
    <selector>
        <enum name="黒" num="0"/>
        <enum name="青" num="1"/>
        <enum name="赤" num="2"/>
        <enum name="緑" num="3"/>
    </selector>
    */
    public int m_type ;

    // <name>向き</name>
    // <sub_type>direction</sub_type>
    public Vector3d m_direction ;

    // <name>速度</name>
    // <range min="0" max="100"/>
    public double m_velocity ;

    // エディタで入力しない変数
    public int m_noname ;

    // <name>エディタで入力しない変数</name>
    protected int m_invisible ;
}
