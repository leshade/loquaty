setProjectionScreen
public native boolean setProjectionScreen( const Vector3* vScreen, double zScale = 1., double fpPixelAspect = 1. )
投影スクリーン座標設定
vScreen
スクリーン中心ｘｙ座標とｚ位置
zScale
投影スケール
fpPixelAspect
アスペクト比
vScreen.z != 0 の時の、座標 (x, y, z) の投影座標 (x', y'):
x' = x * vScreen.z * zScale / z / fpPixelAspect + vScreen.x
y' = y * vScreen.z * zScale / z + vScreen.y

vScreen.z == 0 の時の、座標 (x, y, z) の投影座標 (x', y'):
x' = x * zScale / fpPixelAspect + vScreen.x
y' = y * zScale + vScreen.y
