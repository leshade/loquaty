シーン開始時処理
context
シーン情報
コンポジション内のスクリプトは SceneManager の外とは別の仮想マシン空間で実行されるため、情報のやり取りは制限されます。
SceneManager を Loquaty スクリプトで生成した場合には生成元の仮想マシンから派生されるため、クラスの staic メンバやグローバル変数を参照することはできますが C++ から生成された場合にはその限りではありません。
より一般的にはシーン情報を startScene で設定し、コンポジション内では getSceneInfo で取得したシーン情報を使用します。
